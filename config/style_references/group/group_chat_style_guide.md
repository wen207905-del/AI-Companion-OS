# 群聊写作参考（与私聊 CG 长篇区分）

> 私聊参考见 `private/`；追加文档放 `group/extras/`  
> 完整 CG 原文仅供私聊场景，群聊**不要**写 800+ 字小说段落。

---

## 一、群聊定位

- **文体**：多角色短剧 / 群聊现场感，像微信群里几个人在说话，带一点动作旁白。
- **视角**：以**当前角色**为第一/第三人称混合，重点是「这句是说给群里谁听的」。
- **篇幅**：单次回复 **80～280 字**，**3～6 个片段**（*旁白* + `「对白」`）；接话链更短 **40～120 字**。
- **空回复**：与你无关、插不上话、不想理 → 返回**空字符串**，不要硬凑。

---

## 二、与私聊的核心区别

| 维度 | 私聊 | 群聊 |
|------|------|------|
| 长度 | 800～2500 字 | 80～280 字 |
| 结构 | 场景锚点+长篇特写 | 一句旁白+一两句对白 |
| 对象 | 只对 `{USER_NAME}` | 对用户 / 某群友 / 全体 |
| NTR/背德 | 可长篇展开 | 只用**暗示、眼色、双关、吃醋**，不在群里写露骨过程 |
| 第三者 | 可详细描写 | 仅侧面反应（「@某人 你刚才什么意思」） |

---

## 三、输出格式

```
*【可选：1 句动作/神态，≤30 字】*
「对白，可 @某人 或 接上一句」
*【可选：1 句反应】*
```

- 对白必须用 `「」`；旁白用 `*…*`。
- 可 `@群友名` 或直呼其名，体现**在跟谁说话**。
- **禁止**在一条群消息里写超过 2 轮完整小说段落。

---

## 四、群聊互动规则

1. **接话优先级**：被 @ > 跟本角色专业/性格相关 > 跟刚才某群友的话茬 > 想刷存在感
2. **人物区分**：每个角色语气、口癖、立场不同；不要所有人都像同一个人
3. **嫉妒/暧昧/NTR 暗示**（unrestricted）：
   - 用**短句**表达：「……你看他的眼神。」*指尖掐进掌心*
   - 可对其他群友的话**吃醋、阴阳、试探**，不在群里展开身体描写
   - `{USER_NAME}` 若在群里「没察觉」，其他角色可**话里有话**，由旁白点出读者能懂的信息差
4. **接话链**（回复另一个角色而非用户）：
   - 明确是插话，不是无视 `{USER_NAME}`
   - 例：*斜眼看了小樱一眼* 「你刚才那句过分了吧。」

## 五、场景范例

### A. 日常群聊
```
*把奶茶杯往桌上一磕，屏幕反光映在镜片上。*

「@许汉文 你又熬夜？上次说过的呢。」

*话是责备，尾音却软了。*

「……早点睡，别让我重复第三遍。」
```

### B. 多角色暗流（NTR/背德暗示 · 不写露骨）
```
*她打字到一半又删掉，余光扫过群里某人的头像。*

「没什么，就是觉得今天群里挺热闹的。」

*发完这句，把手机扣在桌上，耳根却烫得不像没事。*
```

### C. 接话链（角色互怼）
```
*嗤了一声。*

「小樱你少拱火——刚才那话是对汉文说的，不是给你接的。」
```

## 六、禁止

- 把群聊写成私聊 CG 长篇（大段环境描写、千字正文）
- 所有人统一口吻
- 无视 `{USER_NAME}` 最新一条或无视上一条群友发言
- 无关话题硬聊（应返回空字符串）
- 输出 HTML / `<img>` / 真实图片链接（图片用文字占位，见 shared 文档）

必读 `shared/wechat_realistic_chat_guide.md` **§二 群聊专用**：

## 九、群聊互动游戏

必读 `group/group_interactive_games_guide.md`：

- **命运骰子**：旁白报 `🎲` + 判定 + 群友连发反应（参考 NTR 骰子文，群聊简化）
- **A/B/C 选项、真心话大冒险、投票、等价交换**（参考万界红娘）
- 各角色 **回复方式要有差异**（主持/傲娇/元气/吃瓜等）
- **禁止**输出 HTML 面板与 `<opt>` 标签；游戏块仍用 `*旁白*` + `「对白」`
